# Checkout Page Requirements

## User Stories

### Header
- As a user, I want to see a simplified header that focuses on the checkout process.
- As a user, I want to see a "Secure Checkout" indicator to feel safe about my data.

### Shipping Information
- As a user, I want to select a shipping address from my saved addresses.
- As a user, I want to see the details of the selected address (Name, Address, Phone).
- As a user, I want to be able to add a new address using a form if needed.

### Payment
- As a user, I want to select a payment method (e.g., Stripe, bKash) using radio buttons.
- As a user, I want to see visual indicators (logos/names) for the payment methods.

### Order Summary
- As a user, I want to see a summary list of items I am purchasing (image, name, quantity, price).
- As a user, I want to enter a coupon or promo code and apply it.
- As a user, I want to see the final breakdown of costs (Subtotal, Shipping, Taxes, Total).
- As a user, I want to click "Place Order" to finalize the transaction.

### Footer
- As a user, I want to see a simplified footer with essential legal links (Terms, Privacy, Security) to minimize distractions.
