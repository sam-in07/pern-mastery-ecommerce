# Login Page Requirements

## User Stories

### Authentication
- As a user, I want to enter my email address so that I can identify my account.
- As a user, I want to enter my password so that I can authenticate.
- As a user, I want to toggle the visibility of my password so that I can ensure I typed it correctly.
- As a user, I want to see a "Forgot password?" link so that I can recover my account if needed.
- As a user, I want to click a "Sign In" button to submit my credentials.

### Navigation
- As a user, I want to see a "Create an account" button so that I can register if I don't have an account.
- As a user, I want to access the main header links (Logo, Cart, etc.) even while on the login page.

### Legal
- As a user, I want to see links to "Terms of Use" and "Privacy Policy" so that I can review the agreements.
