# Signup Page Requirements

## User Stories

### Registration
- As a user, I want to enter my First Name and Last Name so that I can personalize my account.
- As a user, I want to enter my email address to use as my login identifier.
- As a user, I want to enter a password to secure my account.
- As a user, I want to toggle password visibility to check my input.
- As a user, I want to click "Create Account" to finalize my registration.

### Navigation
- As a user, I want to see a link to "Sign In" so that I can log in if I already have an account.
- As a user, I want full access to the site header and footer.

### Legal
- As a user, I want to be informed that by continuing I agree to the Terms of Use and Privacy Policy.
